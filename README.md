- Gitflow

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- git
- Broswer to Display
- IDE to run and edit the code

### Setup

- open a terminal
- run ``` git clone git@github.com:henrycode460/vet_clinic.git```
- run ``` cd vet_clinic```
